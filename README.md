**minibuddy** is our submission to the GDG on Campus Solution Challenge 2025 â€” a cheerful **talk-driven mental health checker** designed for people living alone. Through light, everyday conversations, minibuddy helps users check in on their well-being by providing feedback on anxiety, depression, stress, and memory.

This repository contains the Flutter client for minibuddy, which delivers a friendly voice-based interface and connects to the backend for real-time mental health feedback. It uses Firebase for authentication, in-app notifications, and messaging, and integrates with a TTS API for voice output.
